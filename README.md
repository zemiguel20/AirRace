# AirRace

Small flight race game, kinda like Red Bull Air Race

## Design

*Objective*: Fly a plane through a circuit in the fastest time possible.

*Controls*: Turn the plane with roll, pitch, and yaw axis. Thrust is constant.

*Rules*: Checkpoints in the circuit, and colliding with obstacles makes player respawn at last checkpoint.
